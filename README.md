# TravenManager

TravenManager 是一个为 SillyTavern 设计的配置管理工具，提供精美的 GUI 界面，帮助用户快速便捷地管理 SillyTavern 的各项配置并自动优化其中的数据（如世界书和角色卡）。

## 主要功能

- **一键启动 SillyTavern**：快速启动 SillyTavern 进程
- **实时查看日志**：实时捕获和显示 SillyTavern 启动日志
- **配置管理**：管理 SillyTavern 配置，支持不同 API 模式
- **更新管理**：检查和安装 SillyTavern 更新
- **数据优化**：
  - 世界书管理与优化（编辑、翻译、润色、生成）
  - 角色卡管理与优化（编辑、翻译、润色）
  - 标签管理系统
- **记忆插件系统**：
  - Excel 模板管理
  - 聊天记录管理
  - 表格关联功能
  - 表格整理功能
- **日志系统**：完整的日志分级系统和专用日志面板
- **仪表盘增强**：背景图片功能和统计卡片优化
- **动画系统**：支持各种动画效果
- **紧凑模式**：优化界面空间使用

## 技术栈

- **前端**：React 18, TypeScript, Ant Design 6, Zustand, Vite
- **后端**：Electron, Node.js, WebSocket
- **其他**：ws, zod, @lenml/char-card-reader, crc, json5, mkdirp, rimraf

## 安装与运行

### 环境要求

- Node.js 18.17.0 或更高版本
- npm 11.6.2 或更高版本

### 启动方法

1. **直接启动**：双击 `start.bat` 文件
2. **手动启动**：
   ```bash
   # 安装依赖
   npm install
   
   # 启动开发服务器
   npm run dev
   ```

### 构建方法

```bash
# 构建生产版本
npm run build

# 构建 Electron 应用
npm run electron:build
```

## 核心功能模块

### 1. SillyTavern 进程管理
- 启动、停止 SillyTavern 进程
- 强制终止所有 SillyTavern 相关进程
- 监控 SillyTavern 进程状态

### 2. 配置管理
- 更新 SillyTavern 配置
- 支持不同 API 模式的配置
- 管理模型参数
- 支持配置导入/导出

### 3. 数据优化
- **世界书管理**：编辑、翻译、润色、生成、导入功能
- **角色卡管理**：编辑、翻译、润色、导入功能
- **标签管理**：基于 AI 的标签自动分类系统

### 4. 记忆插件系统
- **Excel 模板管理**：创建和配置 Excel 模板，支持多页签结构
- **聊天记录管理**：树形结构展示，支持分页、搜索、筛选
- **表格关联**：将聊天记录与 Excel 模板关联
- **表格整理**：使用 AI 自动整理聊天记录到表格

### 5. 日志系统
- 支持 4 种日志级别：错误、警告、信息、调试
- 专用日志面板，支持日志过滤和搜索
- 不同级别的日志差异化颜色标识

### 6. AI 功能
- **AI 翻译**：支持单个字段翻译，保留姓名不翻译
- **AI 润色**：支持单个字段润色，支持自定义要求
- **AI 生成**：支持世界书条目生成
- **AI 请求处理**：在主进程中处理 AI 请求，避免 CORS 问题

## 开发规范

- **代码风格**：使用 TypeScript，遵循 ESLint 和 Prettier 规范
- **命名约定**：使用 PascalCase 命名组件，camelCase 命名变量和函数
- **文件结构**：按功能模块组织文件，保持目录结构清晰
- **日志记录**：使用统一的日志系统，记录关键操作和错误
