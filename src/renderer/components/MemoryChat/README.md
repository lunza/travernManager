# 记忆插件系统使用文档

## 概述

记忆插件系统是 TravenManager 的核心功能模块，用于管理 SillyTavern 聊天记录的长期记忆。系统采用以下架构：

- **TravenManager**：管理端核心，提供表格模板管理和聊天记录管理功能
- **SillyTavern + 插件**：编辑输入端，负责聊天交互
- **JSON**：数据持久化存储介质
- **AI 模型及提示词系统**：记录处理引擎，智能提取关键信息

## 功能模块

### 1. 表格模板管理

#### 功能说明
- 创建和配置表格模板
- 支持多页签结构（时空表格、角色表格、社交表格、物品表格、事件表格等）
- 自定义表头字段的增删改查
- 版本控制，保留历史修改记录

#### 使用步骤
1. 在侧边栏点击"记忆管理"进入管理界面
2. 切换到"表格模板管理"页签
3. 点击"新建模板"按钮
4. 输入模板名称和描述
5. 添加页签（如"时空表格"）
6. 为每个页签配置表头字段
7. 点击"保存"完成创建

#### 预设模板
系统默认提供以下模板：
- **默认记忆模板**：包含 5 个基础表格
  - 时空表格：记录时间和空间信息
  - 角色表格：记录角色信息和关系
  - 社交表格：记录社交互动信息
  - 物品表格：记录物品信息
  - 事件表格：记录重要事件

### 2. 聊天记录管理

#### 功能说明
- 完整展示聊天记录
- 支持分页、搜索和筛选功能
- "整理表格"操作，触发 AI 处理流程
- AI 智能提取关键信息并更新至对应 Excel 表格
- 处理结果预览，确认后再执行表格更新

#### 使用步骤
1. 切换到"聊天记录管理"页签
2. 从列表中选择要处理的聊天会话
3. 查看聊天记录详情
4. 选择要使用的 Excel 模板
5. 点击"整理表格"按钮
6. AI 自动分析聊天记录并提取关键信息
7. 预览 AI 处理结果
8. 确认无误后点击"应用结果"

#### 搜索功能
- 支持关键词搜索
- 可在特定会话内搜索或全局搜索
- 搜索结果高亮显示

## 技术架构

### 主进程服务
- **TableTemplateService**：表格模板管理服务
  - 模板的 CRUD 操作
  - 表格文件读写
  - 版本控制和备份

- **ChatLogService**：聊天记录管理服务
  - 聊天会话管理
  - 聊天记录读取和搜索
  - AI 处理流程集成

### 渲染进程组件
- **MemoryTemplateManager**：表格模板管理 UI
- **MemoryChatManager**：聊天记录管理 UI
- **MemoryChatManager**：主管理组件（整合两个子组件）

### IPC 通信
通过 Electron IPC 实现主进程和渲染进程之间的通信：
- `memory:*` 系列 API 提供完整的数据操作接口

## 数据存储

### 目录结构
```
data/
  memory/
    templates/          # Excel 模板文件
      backups/          # 模板版本备份
  chatlog/              # Excel 聊天记录文件
```

### 文件格式
- **模板文件**：JSON 格式，存储在 `data/memory/templates/`
- **聊天记录**：Excel (.xlsx) 格式，存储在 `data/chatlog/`

## AI 处理流程

### 提示词构建
系统自动构建提示词，引导 AI 分析聊天记录并提取以下类别的信息：
1. 时空信息（时间、地点）
2. 角色信息（角色名、身份、关系、特征）
3. 社交互动（时间、参与人、事件、结果）
4. 物品信息（物品名、类型、描述、状态）
5. 事件信息（时间、事件名、参与人、描述、影响）

### 响应解析
AI 返回 JSON 格式的数据，系统自动解析并映射到对应的 Excel 表格。

### 结果预览
提供可视化的结果预览界面，用户可以：
- 查看每个表格的更新内容
- 确认或取消应用更新

## 最佳实践

### 模板设计
1. **明确分类**：根据功能需求设计不同的页签
2. **合理表头**：表头字段应清晰明确，便于 AI 理解
3. **适度粒度**：不要设计过于复杂的表头，保持简洁

### 聊天处理
1. **定期整理**：建议每次对话后进行整理，避免积累过多数据
2. **预览确认**：始终预览 AI 处理结果，确保准确性
3. **模板匹配**：选择与聊天内容匹配的模板进行处理

### 数据管理
1. **版本备份**：定期备份重要模板
2. **清理旧数据**：定期清理不再需要的聊天记录
3. **分类存储**：按项目或角色分类存储 Excel 文件

## 故障排除

### 常见问题

#### 1. AI 处理失败
- **原因**：API 配置错误或网络连接问题
- **解决**：检查配置管理中的 API 设置，确保服务正常运行

#### 2. Excel 文件无法创建
- **原因**：目录权限问题或路径不存在
- **解决**：检查 `data/chatlog/` 目录是否存在并有写入权限

#### 3. 模板加载失败
- **原因**：模板文件损坏或格式错误
- **解决**：从备份恢复模板或重新创建

## 扩展开发
